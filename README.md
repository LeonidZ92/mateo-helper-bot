# Mateo Helper Bot 🤖

Telegram-бот для отслеживания ежедневных рабочих привычек с дневником и умными напоминаниями.

## Возможности

- 🔥 **Серия дней** — отслеживает непрерывную работу с учётом выходных
- 📝 **Дневник** — записывай что сделал за день, смотри историю за 1/7/30 дней
- 🔔 **Умные напоминания** — выбери рабочие дни и время уведомлений
- 🚫 **Сброс серии** — с подтверждением через inline-кнопки

## Стек

- Python 3.10
- aiogram 3.27
- aiosqlite
- systemd (деплой на VPS)

## Запуск

1. Установи зависимости:
```
pip install -r requirements.txt
```

2. Создай файл `.env`:
```
echo "BOT_TOKEN=твой_токен" > .env
```

3. Запусти бота:
```
python bot.py
```

## Деплой на VPS

Бот настроен как systemd сервис с автозапуском:

```
systemctl enable tgbot
systemctl start tgbot
```
